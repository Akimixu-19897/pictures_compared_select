# 婚纱照选片工具 - 原型图设计

## 1. 设计规范

### 1.1 色彩系统

```
主色调:
├── 背景色:      #1A1A1A (深色主背景)
├── 表面色:      #252525 (卡片、面板背景)
├── 表面亮色:    #2D2D2D (悬停状态)
├── 边框色:      #3D3D3D (分割线、边框)

强调色:
├── 主强调色:    #B76E79 (玫瑰金 - 选中、按钮)
├── 主强调hover: #C98992
├── 次强调色:    #F7E7CE (香槟金 - 高亮)
├── 成功色:      #4CAF50 (保留)
├── 警告色:      #FFC107 (提示)
├── 删除色:      #EF5350 (删除、不选)

文字色:
├── 主文字:      #FFFFFF (标题、重要文字)
├── 次文字:      #B0B0B0 (正文)
├── 辅助文字:    #808080 (说明、提示)
└── 禁用文字:    #505050 (不可用状态)
```

### 1.2 字体规范

```
字体栈: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif

字号:
├── H1 - 标题:      24px / font-weight: 600
├── H2 - 副标题:    18px / font-weight: 500
├── Body - 正文:    14px / font-weight: 400
├── Small - 小字:   12px / font-weight: 400
└── Caption - 标注: 11px / font-weight: 400

行高: 1.5
```

### 1.3 间距系统

```
基础单位: 8px

间距:
├── xs:  4px
├── sm:  8px
├── md:  16px
├── lg:  24px
├── xl:  32px
└── xxl: 48px

圆角:
├── sm:  4px  (按钮、标签)
├── md:  8px  (卡片、输入框)
└── lg:  12px (面板、模态框)
```

## 3. 交互动效说明

### 3.1 照片移动动画

```
用户按 Backspace 删除照片:

初始状态:
┌─────┐
│ 📷  │  ← 选中状态
└─────┘

动画过程 (400ms):
1. 照片缩小至 0.8
2. 同时向右上方移动 (飞向回收站方向)
3. 透明度从 1 降至 0
4. 使用 cubic-bezier(0.4, 0, 0.2, 1) 缓动

目标位置 (回收站):
回收站区域显示 "+1" 数字动画
照片从回收站顶部渐显落入
```

### 3.2 分组切换动画

```
点击分组标签:

1. 当前选中标签: 背景色从玫瑰金渐变回深色 (200ms)
2. 新选中标签: 背景色从深色渐变到玫瑰金 (200ms)
3. 主内容区域: 淡出当前照片 (150ms) → 淡入新分组照片 (150ms)
4. 使用 ease-in-out 缓动
```

### 3.3 照片导入动画

```
照片导入完成显示:

1. 缩略图加载完成后，照片卡片从 opacity: 0, scale: 0.9 开始
2. 渐显并放大到正常尺寸 (150ms)
3. 使用 stagger 效果，每张照片延迟 30ms
4. 形成波浪式的入场效果
```

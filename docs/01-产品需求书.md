# 婚纱照选片工具 - 产品需求书 (PRD)

## 1. 项目概述

### 1.1 产品名称
**PhotoPicker Pro** - 专业婚纱照选片工具

### 1.2 产品定位
为婚纱照相馆提供高效、直观的客户选片解决方案，替代传统的纸质选片或文件夹浏览方式，提升客户选片体验和相馆工作效率。

### 1.3 目标用户
- **主要用户**：婚纱照相馆客服人员、选片师
- **终端用户**：前来选片的婚纱照客户（新人）

### 1.4 使用场景
- 客户拍摄完成后，到店内进行照片筛选
- 客服将照片导入系统，客户通过大屏幕浏览并选择精修照片
- 最终输出客户选中的照片列表，供后期精修使用

## 2. 功能需求

### 2.1 核心功能模块

#### 2.1.1 照片导入模块
| 功能点 | 优先级 | 描述 |
|--------|--------|------|
| 文件夹拖拽导入 | P0 | 支持将桌面文件夹直接拖入应用窗口导入 |
| 点击选择导入 | P0 | 通过文件选择对话框选择文件夹导入 |
| 多格式支持 | P0 | 支持 JPG、PNG、RAW 等常见照片格式 |
| 缩略图生成 | P0 | 自动生成分辨率适中的预览缩略图 |
| 导入进度显示 | P1 | 显示扫描和生成缩略图的进度 |

#### 2.1.2 照片浏览模块
| 功能点 | 优先级 | 描述 |
|--------|--------|------|
| 网格视图 | P0 | 以网格形式展示所有照片缩略图 |
| 自适应布局 | P0 | 根据窗口大小自动调整网格列数 |
| 照片信息展示 | P1 | 显示文件名、拍摄时间、文件大小等信息 |
| 缩放功能 | P1 | 支持调整缩略图大小（小/中/大） |
| 快速预览 | P1 | 单击照片进入全屏预览模式 |

#### 2.1.3 照片分组模块
| 功能点 | 优先级 | 描述 |
|--------|--------|------|
| 默认分组（待选） | P0 | 存放所有刚导入的待筛选照片 |
| 保留分组（已选） | P0 | 存放客户确定要精修的照片 |
| 删除分组（不选） | P0 | 存放客户不要的照片 |
| 分组切换 | P0 | 支持点击切换查看不同分组 |
| 分组统计 | P0 | 显示每个分组的照片数量 |

#### 2.1.4 照片操作模块
| 功能点 | 优先级 | 描述 |
|--------|--------|------|
| 选中标记 | P0 | 点击照片选中，显示选中状态 |
| 移入保留组 | P0 | 将照片从默认组移到保留组 |
| 移入删除组 | P0 | 按 Backspace/Delete 移入删除组 |
| 撤销移动 | P0 | 支持撤销上一次的移动操作 |
| 批量操作 | P1 | 支持多选后批量移动 |
| 拖拽移动 | P1 | 支持拖拽照片到不同分组 |

#### 2.1.5 对比功能模块
| 功能点 | 优先级 | 描述 |
|--------|--------|------|
| 双图对比 | P0 | 同时显示两张照片进行对比 |
| 对比源选择 | P0 | 可从任意分组选择照片对比 |
| 跨分组对比 | P0 | 支持默认组 vs 删除组对比 |
| 对比模式切换 | P0 | 左右对比 / 上下对比 |
| 缩放同步 | P1 | 对比时同步缩放和移动位置 |
| 标记决策 | P0 | 在对比界面直接标记保留/删除 |

#### 2.1.6 动画效果模块
| 功能点 | 优先级 | 描述 |
|--------|--------|------|
| 移动动画 | P0 | 照片移动分组时的飞入动画 |
| 选中动画 | P1 | 照片选中时的缩放/边框动画 |
| 删除动画 | P0 | 照片移入删除组时的淡出动画 |
| 导入动画 | P1 | 照片导入时的渐显动画 |
| 切换动画 | P1 | 分组切换时的过渡动画 |

#### 2.1.7 导出功能模块
| 功能点 | 优先级 | 描述 |
|--------|--------|------|
| 保留组导出 | P0 | 导出保留组的照片列表 |
| 生成报告 | P1 | 生成包含照片编号和数量的选片报告 |
| 复制文件 | P1 | 将选中的照片复制到指定文件夹 |
| 数据持久化 | P2 | 保存选片进度，支持断点续选 |

## 3. 非功能需求

### 3.1 性能需求
- 支持单次导入 500+ 张照片
- 缩略图加载时间 < 2秒（100张照片）
- 照片切换延迟 < 100ms
- 内存占用 < 500MB（500张照片场景）

### 3.2 兼容性需求
- 支持 Windows 10/11
- 支持 macOS 11+ (Big Sur 及以上)
- 支持常见屏幕分辨率（1080p - 4K）

### 3.3 用户体验需求
- 界面简洁，学习成本低
- 操作反馈明确
- 动画流畅，不卡顿
- 支持键盘快捷键操作

### 3.4 安全需求
- 不修改原始照片文件
- 支持只读模式浏览
- 操作可撤销，防止误删

---

## 4. 界面需求

### 4.1 界面风格
- **设计风格**：现代简约、专业商务
- **主色调**：深色主题（减少长时间观看疲劳）
- **强调色**：玫瑰金 (#B76E79) 或香槟金 (#F7E7CE)
- **字体**：系统默认字体，保证跨平台一致性

### 4.2 界面布局
- 顶部：工具栏（导入、视图切换、导出）
- 左侧：主照片网格区域（占 70-75%）
- 右侧：删除组区域（占 25-30%）
- 底部：状态栏（照片数量、操作提示）

### 4.3 响应式要求
- 最小窗口尺寸：1280 x 720
- 推荐窗口尺寸：1920 x 1080 及以上
- 支持全屏模式

## 5. 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| `Space` | 选中/取消选中当前照片 |
| `← / →` | 上一张/下一张照片 |
| `↑ / ↓` | 网格中上下移动 |
| `Backspace / Delete` | 移入删除组 |
| `Enter` | 移入保留组 |
| `C` | 进入对比模式 |
| `Esc` | 退出当前模式/取消选择 |
| `Ctrl/Cmd + Z` | 撤销 |
| `Ctrl/Cmd + A` | 全选 |
| `Ctrl/Cmd + I` | 导入文件夹 |
| `Ctrl/Cmd + E` | 导出结果 |

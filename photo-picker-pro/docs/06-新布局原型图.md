## 布局说明

### 1. 左侧 - 默认分组纵列
- **位置**: 最左侧固定宽度纵列
- **宽度**: 约 180-200px
- **内容**: 所有导入的图片缩略图，垂直排列
- **交互**:
  - 点击缩略图 → 在中间区域显示大图
  - 选中状态有视觉反馈（边框高亮）
  - 双击或按空格 → 移动到右侧"不选分组"
- **计数**: 底部绝对定位，显示总数量（如"128张"）

### 2. 中间 - 大图预览区域
- **位置**: 占据中间所有剩余空间
- **功能**: 展示当前选中图片的大图
- **特性**:
  - 图片填满整个区域（object-fit: contain）
  - 鼠标滚轮缩放（保持比例，不扭曲）
  - 支持拖拽平移（放大后）
  - 默认展示第一张图片

### 3. 右侧 - 不选分组纵列
- **位置**: 最右侧固定宽度纵列
- **宽度**: 约 180-200px（与左侧相同）
- **内容**: 被标记为不选的图片缩略图
- **初始状态**: 空的
- **交互**:
  - 双击缩略图 → 移回左侧默认分组
- **计数**: 底部绝对定位，显示数量

### 4. 顶部区域
- **移除**: 原有的分组标签栏（Header）
- **保留**: 最小化的导入按钮（悬浮或角落）

### 5. 底部区域
- **移除**: 原有的状态栏
- **保留**: 两个分组的计数（绝对定位在各自纵列底部）

## 交互流程

```
导入文件夹
    ↓
自动全屏（最大化窗口）
    ↓
左侧显示所有图片缩略图
    ↓
中间自动显示第一张图片大图
    ↓
用户操作:
  ├── 点击左侧缩略图 → 中间显示对应大图
  ├── 双击左侧缩略图/按空格 → 移到右侧不选分组
  ├── 双击右侧缩略图 → 移回左侧默认分组
  ├── 鼠标滚轮 → 缩放中间大图
  └── 按 ESC → 退出全屏
```

## 技术要点

1. **图片缩放**: 使用 CSS transform scale 或自定义实现，保持 aspect-ratio
2. **全屏**: 导入成功后调用 Tauri window API 最大化/全屏
3. **布局**: Flexbox 或 Grid，中间区域 flex: 1
4. **性能**: 虚拟滚动处理大量图片（如有需要）
